왜 활용하기 어려운가?

  • 정보 과잉: 알아야 할 개념이 너무 많고 다양함 (MCP, RAG, 에이전틱 AI, 가상환경 \(\dots\)) — 나에게 필요한 흐름을 잡기 어려움
  • 최적화의 어려움: R도 써야 하고, Python도 써야 하고, LaTeX도 써야 하고… 이 모든 것을 적당히 아우르는 환경을 구축하는 것이 쉽지 않음
  • 상상력의 부족: 에이전틱 AI로 무엇을 할 수 있는지 감을 잡기 어려움

AI 개발 환경과 생태계를 이해하고, 나에게 필요한 환경만 셋팅하는 것도 하나의 역량이라고 생각합니다.

Positron 셋팅: Settings

Cmd + Shift + P → “Open Settings (JSON)” 검색

{
    // Positron 내장 AI 끄기 (Claude Code 사용하므로 불필요)
    "positron.assistant.enable": false,

    // 사이드바 아이콘을 상단에 배치 (화면 넓게 사용)
    "workbench.activityBar.location": "top",

    // 터미널 폰트 (아이콘·특수문자 깨짐 방지)
    "terminal.integrated.fontFamily": "MesloLGS NF",

    // Claude Code를 사이드바에 배치
    "claudeCode.preferredLocation": "sidebar",

    // 마크다운에 이미지 붙여넣기 시 images 폴더에 자동 저장
    "markdown.editor.filePaste.copyIntoWorkspace": "mediaFiles",
    "markdown.copyFiles.destination": {
        "**/*": "images/${fileName}"
    },

    // LaTeX 파일만 자동 줄바꿈
    "[latex]": {
        "editor.wordWrap": "on"
    },

    // R, Python 인터프리터를 자동 실행하지 않고 수동 선택
    "[python]": {
        "interpreters.startupBehavior": "manual"
    },
    "[r]": {
        "interpreters.startupBehavior": "manual"
    }
}
